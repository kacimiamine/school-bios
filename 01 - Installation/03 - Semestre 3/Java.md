## 1. Windows

### 1.1. JDK

#### 1.1.1. Téléchargement

- La plus recommandée est ***Eclipse Temurin 25 LTS***.
- Vous pourrez la trouver ici: ***[Cliquez ici pour télécharger la JDK.](https://adoptium.net/download?link=https%3A%2F%2Fgithub.com%2Fadoptium%2Ftemurin25-binaries%2Freleases%2Fdownload%2Fjdk-25.0.1%252B8%2FOpenJDK25U-jdk_x64_windows_hotspot_25.0.1_8.msi&vendor=Adoptium)***

#### 1.1.2. Installation

- Il suffit juste de faire un double-clic sur le fichier téléchargé.
- Avec la fenêtre qui va afficher, vous cliquez sur **Suivant** jusqu'à la fin.

### 1.2. Eclipse

#### 1.2.1. Téléchargement

- Voici le lien direct pour télécharger Eclipse:
- ***[Cliquez ici pour télécharger](https://www.eclipse.org/downloads/download.php?file=/technology/epp/downloads/release/2025-12/R/eclipse-java-2025-12-R-win32-x86_64.zip&mirror_id=1297)***

#### 1.2.2. Installation

- Ceci est un fichier `zip`. En d'autres termes, un fichier qu'on va extraire.
- Vous pouvez ouvrir ce fichier et cliquer sur **Extraire** ou **Extraire tout** ou n'importe.
- S'il vous demande de choisir l'emplacement, veuillez choisir le disque `C` car c'est plus mieux et plus facile.
- S'il ne vous le demande pas, vous pourrez copier le dossier `eclipse` que vous avez extrait vers le `C`.
- Pour ouvrir **Eclipse**, entrez dans le dossier `eclipse` et cliquez sur `eclipse.exe`.
- Vous pourrez aussi l'envoyer au bureau.
