## 1. Windows

### 1.1. WAMP

#### 1.1.1. Téléchargement

- Vous pourrez le trouver ici: ***[Cliquez ici pour télécharger WAMP.](https://sourceforge.net/projects/wampserver/files/WampServer%203/WampServer%203.0.0/wampserver3.4.0_x64.exe/download)***

#### 1.1.2. Installation

- Il suffit juste de faire un double-clic sur le fichier téléchargé.
- Avec la fenêtre qui va afficher, vous cliquez sur **Suivant** jusqu'à la fin.

### 1.2. VS Code

#### 1.2.1. Téléchargement

- Voici le lien direct pour télécharger VS Code:
- ***[Cliquez ici pour télécharger VS Code](https://code.visualstudio.com/sha/download?build=stable&os=win32-x64)***

#### 1.2.2. Installation

- Avec la fenêtre qui va afficher, vous cliquez sur **Suivant** jusqu'à la fin.
